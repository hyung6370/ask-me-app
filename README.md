### 📌 주요 기능
- Email, Google, Apple Auth 로그인 기능
- STT (Speech To Text) 기능 (레코딩한 말을 텍스트로 바꿔주는 기능)
- OpenAI(ChatGPT) 기능 (말한 질문에 GPT가 대답해주는 기능)
- 대화한 기록들을 저장, 삭제하는 기능

### 📌 개발 환경
- iOS 16.0 +
- Xcode 14.3

### 📌 사용 기술 및 라이브러리
- UIKit
- CocoaPods
- Firebase
- Google STT API
- OpenAI(Chat GPT) API
- Alamofire
- MVC Pattern
